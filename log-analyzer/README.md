
# Log Analyzer Plugin

Analyze any server log using Google Gemini 2.5 Flash AI and get instant, concise solutions for crashes and errors.

## Features
- Analyzes all types of server logs with Google Gemini AI
- Returns short, actionable solutions for detected issues

## Installation
1. Copy the plugin to your `plugins` directory (e.g. `pelican/plugins/log-analyzer`).

## Configuration
1. Create an API key for free in AIStudio:
   - https://aistudio.google.com/api-keys
2. Add your API key to the config via Panel

## Usage
- Open the Log Analyzer tab in your server.
- Click on analyze log.
- The response will be short, direct, and include a solution or a hint if no crash is found.

## Notes
- Do not share your API key publicly.
- The plugin sends log data to the Google Gemini API (see Google privacy policy).
- For support or feedback, use GitHub Issues.
